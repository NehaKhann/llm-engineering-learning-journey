# 📘 Week 1 — LLM Foundations

Welcome to **Week 1** of the **LLM Engineering Learning Journey**.

In this week, you'll explore the fundamental concepts behind Large Language Models (LLMs). Starting with tokenization, you'll progressively build an understanding of how Transformers process language, how modern LLMs generate text, and how different factors influence their behavior.

> **Prerequisite:** Before starting this week, complete the **Neural Networks Fundamentals** lesson located in the `prerequisites/` folder.

---

## 🎯 Learning Objectives

By the end of Week 1, you'll understand:

- How text is tokenized for language models
- How next-token prediction works
- How Transformers use self-attention
- PyTorch fundamentals for deep learning
- How to load and run open-source language models
- How context windows affect model behavior
- How generation parameters influence text generation

## 🎓 Skills You'll Gain

After completing Week 1, you'll be able to:

- Explain how GPT-style models process text
- Understand the Transformer architecture and self-attention
- Work with PyTorch tensors and neural network components
- Load and run pretrained language models
